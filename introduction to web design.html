INTRODUCTION TO WEB DESIGN

DEFINITION OF TERMS

  • INTERNET:
    • Internet is simply defined as the interconnection between computer users.

  • BROWSERS:
    • Browsers are software applications that renders webpages. Example includes Chrome, Uc Browser, Moxila firefox.

  • CLIENT-MACHINE:
    • They are computers that are connected to internet.

  • SERVER:
    • They are dedicated computers that serves other client machines with information as requested and they are always connected to the internet 24/7.

  • HOMEPAGE:
    • It is the very first page that always display when a website is been access.

  • Hypertext markup language(HTML):
    • Hypertext markup language is define as coding language that uses markup method to create hypertext. it is also use in structuring webpage and layout.

  • Cascading stylesheet(CSS):
    • Cascading stylesheet is simply use to style a webpage.

  • JAVA SCRIPT:
    • It is used in adding functionality to webpage.

  • ATTRIBUTE:
    • Attribute is used to change default settings or appearance in a tag and it can be used only in opening tags.

  • TAGS:
    • Tags are command synta that tell the computer what to do. The types includes Container tag and Empty or self closing tag.

  • TEXT EDITORS
    • Text editors are use to write and edit source code. Example include Atom, Sublime, Notepad ++.

  • PROTOCOL:
    • They are set of rules establishes or guides Intermachine communication.

  • UNICODE:
    • They are sets of characters known to the computer.
